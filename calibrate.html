NAVINCLUD - Calibração de Monitor

Instruções obrigatórias antes de iniciar o teste:

  1. Ajuste a temperatura de cor (ponto branco) do monitor para D65 (6500K) usando o menu do monitor.
  2. Ajuste a gama (gamma) para 2.2 (padrão sRGB).
  3. Defina a luminância para 80–120 cd/m² (brilho médio).
  4. Desative modos dinâmicos: brilho automático, contraste dinâmico, modo noturno e economia de energia.
  5. Restaure as configurações de fábrica se necessário e ajuste os parâmetros acima.

Esses ajustes garantem rigor científico e validade ao teste de Ishihara digital.

Teste 3: Textos de Baixo Contraste

Texto cinza escuro (#333) em fundo cinza claro (#ccc) - deve ser legível.
Texto cinza claro (#ccc) em fundo cinza escuro (#333) - deve ser legível.

Teste 4: Texto Pequeno (10px)

Este é um texto de 10 pixels. Ele deve ser perfeitamente legível para garantir que a resolução e nitidez do monitor estão adequadas para o teste.

Validação: Todos os elementos acima devem estar claramente visíveis para prosseguir. Se algum elemento estiver ilegível ou com cores distorcidas, recalibre o monitor antes de iniciar o teste.
