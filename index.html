Install Extension Learn More

Features

Everything you need for a more inclusive web experience.

🔬

Chromatic Screening Simulator (Based on the Ishihara Test)

Pedagogical tool composed of 18 visual perception plates for exploration and chromatic agility testing purposes. Ideal for pattern recognition exercises and color perception training.

🎨

8 Correction Filters

Dynamic filters for protanopia, deuteranopia, tritanopia, achromatopsia and their variations, with adjustable intensity.

📊

Scientific Calibration

Monitor calibration guide with D65 standard, Gamma 2.2 and integrated visual tests for maximum accuracy.

🌐

Browsing Experience

2-minute experience mode that demonstrates how filters transform web browsing in practice.

📥

Data Export

Secure export of results with multi-step confirmation to ensure user control over their data. Data export is active only for the TCC presentation. After the presentation, the data extraction button will be removed from new versions.

🔒

100% Local & Private

All data processed and stored locally in the browser. Nothing is sent to external servers.

How It Works

1.

Calibrate

Adjust your monitor following the scientific calibration guide for accurate results.

2.

Test

Take the digital Ishihara test with 18 plates and discover your color vision type.

3.

Filter

Apply the recommended filter or customize intensity and adjustments to your needs.
