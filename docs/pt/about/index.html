Sobre o NAVINCLUD

Tornando a web mais acessível para todos.

O que é o NAVINCLUD?

NAVINCLUD é uma extensão Chrome que ajuda pessoas com deficiência de visão de cores (DVC) a navegar na web com mais conforto. Combina um teste de percepção de cores baseado no método Ishihara com filtros de correção cromática em tempo real.

Se você tem protanopia, deuteranopia, tritanopia, acromatopsia ou qualquer de suas variações, o NAVINCLUD pode ajudar a tornar as cores mais distinguíveis em qualquer site que você visitar.

Como Usar

  1. Calibre seu monitor — Use o guia de calibração integrado para garantir reprodução precisa de cores.
  2. Faça o teste Ishihara — Identifique números em 18 placas de cores. O teste leva cerca de 2-3 minutos.
  3. Aplique o filtro — Se o teste indicar uma deficiência, a extensão recomenda e aplica automaticamente o filtro mais adequado.
  4. Ajuste conforme necessário — Refine intensidade e shift de cor pelos controles do popup a qualquer momento.

Perguntas Frequentes

O NAVINCLUD é uma ferramenta de diagnóstico médico?

Não. O NAVINCLUD é uma ferramenta interativa para exploração de percepção de cores. Não substitui exames oftalmológicos ou diagnóstico médico profissional.

A extensão coleta meus dados?

Não. Todos os dados são armazenados localmente no seu navegador. Nada é enviado para servidores externos. Veja nossa Política de Privacidade para detalhes.

Quais tipos de daltonismo são suportados?

NAVINCLUD suporta protanopia, protanomalia, deuteranopia, deuteranomalia, tritanopia, tritanomalia, acromatopsia e acromatomalia — 8 tipos de filtro no total.

A extensão é gratuita?

Sim, o NAVINCLUD é completamente gratuito e open source sob licença MIT.

Posso ajustar a intensidade do filtro?

Sim. O popup inclui controles deslizantes para intensidade e shift de cor, além de um botão liga/desliga para comparar rapidamente a visualização com e sem filtro.
