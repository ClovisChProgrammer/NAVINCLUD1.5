Funcionalidades

Tudo que você precisa para uma experiência web mais inclusiva.

🔬

Teste de Percepção de Cores (Método Ishihara)

Teste interativo com 18 placas Ishihara para identificar o tipo de visão de cores. Resultado instantâneo com recomendação automática de filtro.

🎨

8 Filtros de Correção

Filtros dinâmicos para protanopia, deuteranopia, tritanopia, acromatopsia e suas variações, com ajuste de intensidade e shift de cor.

📊

Calibração de Monitor

Guia de calibração integrado com padrão D65, Gamma 2.2 e testes visuais para máxima precisão de cores.

🌐

Filtros em Tempo Real

Os filtros são aplicados instantaneamente em qualquer site que você visitar, adaptando as cores para uma navegação fluida.

🧩

Recomendação Automática

Com base no resultado do teste Ishihara, a extensão sugere e aplica automaticamente o filtro mais adequado para você.

🔒

100% Local e Privado

Todos os dados processados e armazenados localmente no seu navegador. Nada é enviado para servidores externos. Sua privacidade protegida.
