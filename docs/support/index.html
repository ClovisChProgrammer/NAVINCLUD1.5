Chrome Web Store (Em Breve)

The extension is coming soon to the Chrome Web Store. Check back for the installation link.

Manual Installation (Developer Mode)

If you want to try the extension before the CWS release:

  1. Download or clone the repository from GitHub
  2. Open Chrome and go to chrome://extensions/
  3. Enable Developer Mode (top right)
  4. Click Load unpacked and select the extension folder
  5. The NAVINCLUD icon will appear in your toolbar
