About NAVINCLUD

Making the web more accessible for people with color vision deficiencies.

The Problem

Approximately 350 million people worldwide have some form of color blindness or achromatopsia. The spectrum of color vision deficiencies includes:

  • Protanopia/Protanomaly — difficulty with red tones
  • Deuteranopia/Deuteranomaly — difficulty with green tones
  • Tritanopia/Tritanomaly — difficulty with blue tones
  • Achromatopsia/Achromatomaly — partial or total grayscale vision

Despite being a common condition, most websites and applications lack proper support, excluding a significant portion of the population.

The Solution

NAVINCLUD is a free and open-source Chrome extension that offers:

  • Diagnostic test based on the Ishihara method with 18 plates to identify the type of color vision deficiency
  • Real-time correction filters applied via SVG feColorMatrix on any web page
  • Scientific monitor calibration to ensure accurate results
  • Browsing experience to demonstrate the impact of filters
  • Total privacy — all data processed locally

Technology

  • Manifest V3 — the latest and most secure Chrome extension API
  • SVG feColorMatrix — color filters applied via transformation matrix on the DOM
  • Chrome Storage API — secure local storage of results
  • WebP — optimized Ishihara plates for fast loading

Academic Project

NAVINCLUD was developed as a Course Completion Project at ETEC Comendador João Rays (Barra Bonita/SP, Brazil), under the guidance of the institution's professors.
