Sobre o NAVINCLUD

Tornando a web mais acessível para pessoas com deficiência cromática.

O Problema

Aproximadamente 350 milhões de pessoas no mundo têm alguma forma de daltonismo ou acromatopsia. O espectro de deficiências cromáticas inclui:

  • Protanopia/Protanomalia — dificuldade com tons de vermelho
  • Deuteranopia/Deuteranomalia — dificuldade com tons de verde
  • Tritanopia/Tritanomalia — dificuldade com tons de azul
  • Acromatopsia/Acromatomalia — visão em escala de cinza parcial ou total

Apesar de ser uma condição comum, a maioria dos sites e aplicações não oferece suporte adequado, excluindo uma parcela significativa da população.

A Solução

NAVINCLUD é uma extensão gratuita e open-source para Chrome que oferece:

  • Teste diagnóstico baseado no método Ishihara com 18 placas para identificar o tipo de deficiência cromática
  • Filtros de correção em tempo real aplicados via SVG feColorMatrix em qualquer página web
  • Calibração científica de monitor para garantir precisão nos resultados
  • Experiência de navegação imersiva para demonstrar o impacto dos filtros
  • Privacidade total — todos os dados processados localmente

Tecnologia

  • Manifest V3 — API mais moderna e segura para extensões Chrome
  • SVG feColorMatrix — filtros de cor aplicados via matriz de transformação no DOM
  • Chrome Storage API — armazenamento local seguro dos resultados
  • WebP — placas Ishihara otimizadas para carregamento rápido

Projeto Acadêmico

O NAVINCLUD foi desenvolvido como Trabalho de Conclusão de Curso na ETEC Comendador João Rays (Barra Bonita/SP), sob orientação dos professores da instituição.
