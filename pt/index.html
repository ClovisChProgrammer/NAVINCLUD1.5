Funcionalidades

Tudo que você precisa para uma experiência web mais inclusiva.

🔬

Simulador de Triagem Cromática (Baseado no Teste de Ishihara)

Ferramenta pedagógica composta por 18 pranchas de percepção visual para fins de exploração e teste de agilidade cromática. Ideal para exercícios de reconhecimento de padrões e treinamento de percepção de cores.

🎨

8 Filtros de Correção

Filtros dinâmicos para protanopia, deuteranopia, tritanopia, acromatopsia e suas variações, com ajuste de intensidade.

📊

Calibração Científica

Guia de calibração do monitor com padrão D65, Gamma 2.2 e testes visuais integrados para máxima precisão.

🌐

Experiência de Navegação

Modo experiência de 2 minutos que demonstra na prática como os filtros transformam a navegação web.

📥

Exportação de Dados

Exportação segura dos resultados com confirmação em múltiplas etapas para garantir o controle do usuário. A exportação de dados está ativa apenas para a apresentação do TCC. Após a apresentação, o botão de extração de dados será retirado das novas versões.

🔒

100% Local e Privado

Todos os dados processados e armazenados localmente no navegador. Nada é enviado para servidores externos.
