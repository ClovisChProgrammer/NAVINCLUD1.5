Política de Privacidade

Última atualização: maio de 2026

Resumo

O NAVINCLUD respeita sua privacidade. Esta extensão foi projetada com o princípio de privacidade por padrão — nenhum dado pessoal é coletado, armazenado em servidores ou compartilhado com terceiros.

Dados Coletados

A extensão coleta e armazena exclusivamente no dispositivo do usuário:

  • • Resultados do teste Ishihara — placas visualizadas, tempos de resposta, pontuação e diagnóstico gerado. Utilizados apenas para exibir o resultado ao usuário.
  • • Preferências de filtro — tipo de filtro selecionado, intensidade e ajuste de shift. Utilizados para aplicar a correção visual personalizada.
  • • Identificador único de teste — UUID gerado aleatoriamente para organizar o histórico local de testes. Não permite identificação do usuário.

Armazenamento

  • • Todos os dados são armazenados exclusivamente no chrome.storage.local — uma área de armazenamento interna do navegador, isolada e não acessível a outros sites ou extensões.
  • • Nenhum dado é transmitido para servidores remotos, APIs externas ou terceiros.
  • • Os dados permanecem no dispositivo até que o usuário opte por exportá-los ou removê-los.

Dados que NÃO Coletamos

  • ✖ Dados pessoais identificáveis — nome, email, CPF, endereço, etc.
  • ✖ Dados de navegação — histórico, URLs visitadas (exceto quando o filtro está ativo para aplicar a correção visual)
  • ✖ Credenciais — senhas, tokens, cookies de autenticação
  • ✖ Analytics ou rastreadores — a extensão não utiliza Google Analytics, telemetria ou qualquer serviço de monitoramento

Compartilhamento com Terceiros

O NAVINCLUD não compartilha nenhum dado com terceiros. Não há integração com APIs externas, serviços de publicidade ou redes sociais.

Direitos do Usuário (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), você tem os seguintes direitos:

  • Confirmação e acesso — visualize seus dados de teste no popup da extensão
  • Correção — realize um novo teste para atualizar seus resultados
  • Exclusão — exporte os dados e limpe o armazenamento local pelo popup
  • Portabilidade — exporte seus testes em formato JSON
  • Transparência — esta política detalha exatamente como os dados são tratados

Segurança

A extensão opera com as permissões mínimas necessárias e todo o processamento é feito localmente. O acesso ao armazenamento local é isolado pelo próprio navegador, seguindo o modelo de segurança do Manifest V3.

Alterações nesta Política

Esta política pode ser atualizada para refletir mudanças na extensão ou requisitos legais. A data da última atualização está no topo da página. Recomendamos revisá-la periodicamente.
